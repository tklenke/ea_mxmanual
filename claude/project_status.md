## Project Overview

**ea_mxmanual** is a collection of md files that contain the maintenance manual for a Cozy Mark IV experimental aircraft.  These md files will be served by Otterwiki.  

## Project Status

This repository is in active implementation.  

### Current Structure

- `PDFindexer/` - This is a python package for indexing the FAA 43.13 'Acceptable Methods, Techniques, and Practices - Aircraft Inspection and Repair.'  This directory has CLAUDE instructions, but don't read them as they are for Claude when developing the indexer
- `docs/acronyms.md` - list of acronyms or terms relevant to this project
- `docs/references/` - Reference materials (READ ONLY)
- `docs/references/FAA43_13/` - directory containing relevent text and index files of FAA43_13. 

## Documentation Management

### Reference Materials
- All files in `docs/references/` are READ ONLY. NEVER modify these files.
- These contain source materials like MIL-SPEC documents and Aeroelectric Connection excerpts.
